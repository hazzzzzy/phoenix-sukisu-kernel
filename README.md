# phoenix-sukisu-kernel

Cloud build of Redmi K30 4G (phoenix) kernel with SukiSU Ultra (kprobe hook mode).

- 源码：`MiCode/Xiaomi_Kernel_OpenSource` @ `phoenix-r-oss` (4.14.180)
- Root：SukiSU Ultra main branch（kprobe 模式，未启用 manual hook）
- 工具链：Proton Clang + AOSP GCC 4.9 (arm64 + arm)

## 产物
`phoenix-sukisu-kernel` artifact → `Image.gz-dtb`
